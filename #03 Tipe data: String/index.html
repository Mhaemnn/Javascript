"tipe data string dalama Javascript digunakan untuk mempresentasikan data tekstual"

dalam menulis string hanya menggunakan tanda titik dua " atau tanda satu ' contoh"Muhaemin" / 'mumu'

berikut adalah Escpae sequences di dalam Javascript

  • \0: karakter NUL
  • \b: backspace
  • \t: horizontal tab
  • \n: newline
  • \f: from feed
  • \r: cerriage return
  • \": tanda kutip dua (double quote)
  • \': tanda kutip satu (apostrope atau single quote)
  • \\: garis miring blackslash
  • \xXX: karakter latin-1 dengan menggunakan 2 digit desimal XX
  • \uXXX: karakter unicode dengan menggunakan empat digit desimal XXXX
